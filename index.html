⚠ Gunakan F-Droid, bukan Play Store — versi Play Store Termux sudah tidak diupdate dan tidak kompatibel.
2 Buka Termux, jalankan perintah ini

Salin perintah di bawah, tempel di Termux, lalu tekan Enter. Installer akan berjalan otomatis dan menanyakan akun kamu.

# Ganti YOUR_GITHUB_TOKEN dengan token kamu
curl -sSL https://raw.githubusercontent.com/ShadowSoldiers/xeatools/main/setup.sh | bash -s YOUR_GITHUB_TOKEN
✓ Ganti YOUR_GITHUB_TOKEN dengan token GitHub kamu. Installer akan meminta username & password XEA serta akun Gmail — tidak perlu edit file apapun secara manual.
3 Atau scan QR Code ini di HP lain alternatif step 2

Scan dengan kamera HP → buka Termux → tempel → Enter.
QR ini berisi perintah curl di atas.

4 Setelah instalasi selesai
Jalankan Web GUI dengan perintah: cd ~/merge_pdf && bash start_manual.sh
Buka Chrome Android → ketik localhost:5000
Tab Download → masukkan rentang tanggal → unduh dokumen Akun XEA sudah terkonfigurasi saat install
Tab Merge → jalankan merge → konfirmasi kirim email
